# Robot scoring system for the compentition

A Java console application to manage robots
compentition using OOP principles.

## Features
- Display all information for LineFollower and RoboSoccer objetcs
- Sort objects according to their age group and the according to score , using Comparable interface

  # Technologies Used
  - Java
  - OOP(Inheritance , polymorphism)
  - Arrays
  - File I/O
