# Robot scoring system for the compentition

A Java console application to manage robots
compentition using OOP principles.

The SANRA (south African National Robotics Association) is hosting a robotics competition in 2026.
There is a line following competition as well as a robo-soccer competition. So im in a team developing the scoring system for the competition

For the line following competition competitors need to build and program automous robots that will follow a line on the playing field.
(An automous robot is a robot thta operates on it's own withouth being controlled by an operator.) The total distance that the robot successfully followed the line as well
as the time that it took is recorded in this competition. As soon as the robot loses the line its run is stopped and the distance and time is recorded

Robots in the robo-soccer competition are remotely controlled by competitors. The aim in this competition is to collect as many balls in the playing field
as possible and take them to the robot's goal area. In each match two robots , each with its own goal area compete in the same game area. Robots are not alowed to bump into
each other. The robot that is guilty of bumping into the other is given a penalty.

## Features
- Display all information for LineFollower and RoboSoccer objetcs
- Sort objects according to their age group and the according to score , using Comparable interface
